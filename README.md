# R-Type (MVP) - Browser Canvas

Questo è uno scaffold per un MVP giocabile in puro JavaScript + HTML5 Canvas.

Come eseguire:

1. Apri `index.html` nel browser (oppure esegui un server locale):

   ```sh
   python -m http.server 8000
   # quindi apri http://localhost:8000
   ```

2. Controlli:
   - Movimento: frecce / WASD
   - Sparo rapido: Z / X / C (tenere premuto per fuoco continuo)
   - Carica Wave Cannon: Space (tieni premuto e rilascia)
   - Toggle Force: F

Note:
 - Questo è un prototipo minimale: collisioni AABB semplici, pooling basico per proiettili, forza semplificata.
 - Prossimi passi: polire collisioni, aggiungere livelli, grafica e audio.
